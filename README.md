## Cómo Usar config

Copiad el bashrc.template.sh en `~` y renombrad a `.bashrc` (haced un backup primero). 

Comandos:

```
  cd ~
  cp .bashrc .bashrc.bak
  mv <ruta a bash_history.sh> .
  source .bashrc
```
